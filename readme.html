Инструкция

  1. Установка библиотек:
    2. Запусти install.bat
    Подожди установки
    Проверь наличие ошибок
  2. Добавление аккаунтов:
    3. Зайди в директорию data/accounts
    Добавь туда аккаунты в формате двух файлов для каждого аккаунта(.maFile, .txt)
  3. Добавление почт firstmail:
    4. Зайди в директорию data/
    Открой файл emails.txt
    Добавь почты(новая почта - новая строка)
  4. Добавление токена firstmail:
    5. Перейди на сайт firstmail
    Нажми в правом верхнем углу История покупок & api
    Зарегистрируйся под своей почтой
    Зайди в раздел API
    Выбери тип ключа: Авторизация в аккаунты..., разрешенный ip: ip на котором происходит ретрив
    Зайди в config.ini и вставь токен после "="
  5. Старт
    6. Запускаешь SDA и start.bat
    После открытия браузера сразу же открываешь SDA на передний план
Аккаунты будут сохраняться в data/output.txt с новой почтой и паролем от нее
Все ошибки будут сохраняться в logs.log
